**Slay the Spire 2 stats program**

A messy, personal stats project built around a game I've played quite a bit recently.

Bootstapping, t-tests, linear and logistic regression, naive bayes, permutation tests - the ususal.

Disclaimer: This was quickly made primarily for learning purposes - and is not an actual good program by any means!

To use the program, just dump all your .run files in a /runs folder, and run:
pip install -r requirements.txt
python slay_the_stats2.py

Code is messy and very unpolished. There are probably a few bugs. Works on my machine though;)
